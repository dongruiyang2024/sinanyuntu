# Sinan Cloudmap Website Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a polished Next.js single-page corporate website for 司南云图（杭州）科技有限公司 using the approved brand assets, brand colors, and “可信导航型” visual direction.

**Architecture:** The app uses Next.js App Router with a static homepage assembled from focused React components. Page copy and repeated section data live in `data/home.ts`, while visual components stay in `components/` so the homepage remains easy to scan and future pages can reuse the header, footer, and brand visuals.

**Tech Stack:** Next.js, React, TypeScript, Tailwind CSS, ESLint, local SVG assets in `public/brand/`.
